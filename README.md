## Installation

<p>env파일 셋팅</p>

```bash
.env파일 생성법

# terminal에서 .exampleEnv파일을 .env파일로 복사
cp .exampleEnv .env

# .env파일 수정
NEXT_PUBLIC_BACKEND_URL=백엔드 기본 URL
NEXT_PUBLIC_TEAM=팀 코드
NEXT_PUBLIC_KAKAO_APP_KEY= 카카오 API키
NEXT_PUBLIC_KAKAO_REDIRECT_URI= 카카오 리다이렉트 URL
```
